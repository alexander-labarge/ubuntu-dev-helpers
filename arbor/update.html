Next Steps

This upload module establishes the foundation for automated raw capture storage with structured metadata (detected bounding box locations) and enables Label Studio integration for annotating edge cases—weapons, prone positions, partial concealments—to train supplementary model weights.

Dashboard v2 & Spectre Inference Engine

The module integrates with Dashboard v2, which runs the Spectre inference engine (our YOLO-based solution) during live optic capture. This enables simultaneous capture, detection, annotation, and metadata extraction—with a side-by-side view of raw thermal feed versus detected humans. All results write directly to the database, feeding the training pipeline.

The first targeted model: rifle detection. Additionally, based on the substantial temperature contrast data collected, we now have 1,000+ labeled data points to train YOLO to discern between standard camouflage and MSC-type thermal suppression equipment.

The upload module will be reused in Dashboard v2 via a new "Upload" tab running the Python server with a pre-shared GitLab key. We can later swap the IP and key for a BSI cloud instance to enable multi-site backup once we iron out the kinks.
